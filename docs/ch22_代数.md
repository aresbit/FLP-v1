## 一、本章概要 (Overview)

- **核心问题**: 从"什么是数"出发，经历加法、乘法、幂运算及其逆运算，逐步将数的概念从正整数推广到负数、分数、无理数、复数，最终抵达"整个数学中最卓越、几乎令人震惊的公式"——欧拉公式 $$e^{i\theta} = \cos\theta + i\sin\theta$$。费曼将这颗"瑰宝"镶嵌在初等代数的宏伟大厦之中，展示的是：人类如何从一个微小的起点——只知道整数和计数——通过反复的"抽象与推广"，自己制造出对数表、幂表乃至三角函数。
- **逻辑链条**: 从计数出发定义加法 → 重复加法得乘法 → 重复乘法得幂 → 定义逆运算（减法、除法、开方、对数）→ 发现某些方程在原数系中不可解 → 通过"保留代数规则、扩展符号含义"的方式推广数系（正整数 → 负整数 → 有理数 → 无理数）→ 用平方根迭代逼近无理数幂和对数 → 发现 $$i = \sqrt{-1}$$ 发明复数 → 用平方/迭代计算复数的虚幂 → 发现周期性 → 代数余弦和代数正弦 → 欧拉公式 → 代数与几何的统一。
- **本章的特殊性**: 这不是物理课——费曼称之为"智识的享受"。他开篇就自问:"数学在物理课上做什么？"然后给出三层回答：第一层是功利性的（数学是有用的工具），第二层是物理性的（物理定律具有数学形式），第三层才是真话——"这个主题很有乐趣"。他从物理学家而非数学家的视角展开：数学家关心"如何证明"以及"最少需要哪些公理"，而费曼关心的是"结果"和"这幅宏大图景的总体面貌"。

### 2.2 加法与乘法 (Addition and Multiplication) —— 22-1

费曼"从中间开始"：假设我们已知什么是整数、什么是 $$0$$、以及"将一个数增加一个单位"是什么意思。如果我们从一个整数 $$a$$ 出发，连续计数 $$b$$ 次，每次增加 $$1$$，我们到达的数称为 $$a + b$$。这就定义了**整数的加法**。

一旦有了加法，我们就可以考虑：从零出发，连续加上 $$a$$，重复 $$b$$ 次——结果称为**整数的乘法**，称为 $$b$$ 乘以 $$a$$。

我们还可以有**连续的乘法**：从 $$1$$ 出发，连续乘上 $$a$$，重复 $$b$$ 次——这就是**求幂**：$$a^b$$。

由这些定义出发，可以轻松证明以下关系全部成立：

$$a + b = b + a,$$

$$a + (b + c) = (a + b) + c,$$

$$ab = ba,$$

$$a(b + c) = ab + ac,$$

$$(ab)c = a(bc),$$

$$(ab)^c = a^c b^c,$$

$$a^b a^c = a^{(b+c)},$$

$$(a^b)^c = a^{(bc)},$$

$$a + 0 = a,$$

$$a \cdot 1 = a,$$

$$a^1 = a. \tag{22.1}$$

其中 $$1$$ 和 $$0$$ 具有特殊性质。费曼也提醒我们还需要假定另外一些性质（如连续性和序关系），但"让严格的理论去处理这些"。此外，确实有可能某些"规则"可以从其他规则推导出来，但"我们不去操心这件事"。

### 2.3 逆运算 (The Inverse Operations) —— 22-2

除了加法、乘法和求幂这些**直接运算**之外，还有**逆运算**，定义如下：假设已知 $$a$$ 和 $$c$$，希望找到满足 $$a + b = c$$、$$ab = c$$、$$b^a = c$$ 的 $$b$$ 值。

正运算与逆运算的对应关系：

| 正运算 | 逆运算 |
|------|------|
| 加法: $$a + b = c$$ | 减法: $$b = c - a$$ |
| 乘法: $$ab = c$$ | 除法: $$b = c / a$$ |
| 幂: $$b^a = c$$ | 方根: $$b = \sqrt[a]{c}$$ |
| 幂: $$a^b = c$$ | 对数: $$b = \log_a c$$ |

由于 $$b^a$$ 和 $$a^b$$ 不相等，幂运算有两个逆运算。求对数的记号虽然比其他的笨重，但应用于整数时并不比求方根更"基本"——它只是代数方程的一种不同类型的解。

### 2.4 抽象与推广 (Abstraction and Generalization) —— 22-3

当我们尝试用正整数解所有代数方程时，立刻遇到了不可解的问题。例如：$$b = 3 - 5$$。在正整数中，不存在这样的数。然而，整个大思想的方案是：**抽象与推广**。

从"代数规则 + 整数"的整个结构中，我们**抽掉**加法和乘法的原始定义，但**保留**规则 (22.1) 和 (22.2)，并假定这些规则在一个更广的数类上普遍成立。于是，我们不再用整数来符号性地定义规则，而是**用规则来定义符号**——这些符号现在代表了一种更一般的数。

例如，我们定义一组新数: $$0 - 1$$, $$0 - 2$$, $$0 - 3$$, $$0 - 4$$, ...，称为**负整数**。然后利用所有其他规则（如 $$a(b + c) = ab + ac$$）来发现负数乘法的规则。一切规则对正整数和负整数都能保持成立——尽管你不能说"$$-2$$ 乘以 $$5$$ 意味着连续加 $$5$$ 加 $$-2$$ 次"，那没有意义。但无论如何，按规则来做，一切都会运行正常。

**分数的引入**同理：不存在整数（即使是负整数）等于 $$3$$ 除以 $$5$$。但如果假设所有分数也满足这些规则，那么分数的加减乘除就像以前一样好用。

**无理数**的引入来自另一个不可解方程: $$b = 2^{1/2} = \sqrt{2}$$。不存在任何一个有理数（分数）的平方等于 $$2$$。费曼指出，从历史看，这一概念给希腊人带来了极大的困难。戴德金 (Dedekind) 后来在形式上和严格性上真正做到了精确的定义。但就我们的目的而言，理解这一点就足够了：我们打算找一整个序列的近似分数——越来越精确——来逼近我们想要的结果。

**费曼的关键方法论**: 规则的普适性。每当我们发现一个在原数系中无解的问题时，我们不是放弃规则，而是**扩展数的领域**，让旧规则在新领域中依然成立。新符号的含义变了，但代数结构不变。

**负指数与分数指数的定义** (规则推出来的定义):
- 从 $$a^{(3-5)} a^5 = a^3$$ 推导出 $$a^{(3-5)} = a^3 / a^5 = 1 / a^2$$ ——所以 **负指数即倒数的正指数**。
- 从 $$(a^{3/5})^5 = a^3$$ 推导出 $$a^{3/5} = \sqrt[5]{a^3}$$ ——所以 **分数指数即方根**。

这些定义不是任意的；我们利用规则本身帮助我们确定定义。

### 2.5 逼近无理数 (Approximating Irrational Numbers) —— 22-4

接下来是实际计算的问题：如何计算像 $$10^{\sqrt{2}}$$ 这样的无理数次幂？原则上简单：将 $$\sqrt{2}$$ 近似到某位小数，则幂变为有理数幂，可用前述方法求近似根。但涉及的方根次幂极高，工作量巨大。因此需要**对数表**。

此外，解 $$10^x = 2$$ 即 $$x = \log_{10} 2$$ 也是一个纯计算问题，答案只是一个无理数，而非新类型的数。

**计算对数表的核心方法**: 连续开平方根。

**表 22-1: 10 的逐次平方根**

| 指数 $$s$$ | $$1024s$$ | $$10^s$$ | $$(10^s - 1) / s$$ |
|---|---|---|---|
| $$1$$ | $$1024$$ | $$10.00000$$ | $$9.00$$ |
| $$1/2$$ | $$512$$ | $$3.16228$$ | $$4.32$$ |
| $$1/4$$ | $$256$$ | $$1.77828$$ | $$3.113$$ |
| $$1/8$$ | $$128$$ | $$1.33352$$ | $$2.668$$ |
| $$1/16$$ | $$64$$ | $$1.15478$$ | $$2.476$$ |
| $$1/32$$ | $$32$$ | $$1.074607$$ | $$2.3874$$ |
| $$1/64$$ | $$16$$ | $$1.036633$$ | $$2.3445$$ |
| $$1/128$$ | $$8$$ | $$1.018152$$ | $$2.3234$$ |
| $$1/256$$ | $$4$$ | $$1.0090350$$ | $$2.3130$$ |
| $$1/512$$ | $$2$$ | $$1.0045073$$ | $$2.3077$$ |
| $$1/1024$$ | $$1$$ | $$1.0022511$$ | $$2.3051$$ |

观察最后一列 $$(10^s - 1)/s$$，随着 $$s$$ 变小，它逼近常数 $$2.3025$$。这揭示了一条关键近似：对于极小量 $$\epsilon$$，

$$10^{\epsilon} \approx 1 + 2.3025\epsilon.$$

**计算 $$\log_{10} 2$$** (表 22-2):

基本思路是将 $$2$$ 分解为表中数值的乘积。由表 22-2 的运算链:

$$2 = (1.77828)(1.074607)(1.036633)(1.0090350)(1.000573).$$

每一项对应于一个条目在表中的指数（以 $$1/1024$$ 为单位）。因此:

$$2 = 10^{\frac{256+32+16+4+0.254}{1024}} = 10^{\frac{308.254}{1024}}.$$

所以 $$\log_{10} 2 = 308.254 / 1024 = 0.30103$$，精确到五位！

这正是哈利法克斯的布里格斯先生 (Mr. Briggs) 在 1620 年最初计算对数的方法——他连续算了 54 次平方根（实际上只算了 27 次，剩下的用 $$\Delta$$ 的规律推断），用十六位小数计算，发表时缩为十四位。此后三百年的对数表都是从布里格斯的表截取小数位数得到的。

**自然底 $$e$$**: 从近似式 $$10^{\epsilon} \approx 1 + 2.3025\epsilon$$，通过缩放变为:
$$e^n \approx 1 + n$$（当 $$n \to 0$$），即 $$\log_e(1 + n) \approx n$$。

$$e$$ 本身是多少？$$e = 10^{1/2.3025\ldots} = 10^{0.434310\ldots}$$。将 $$0.434310\ldots$$ 转化为 $$444.73/1024 = (256+128+32+16+8+4+0.73)/1024$$，用表中数据乘积:

$$e = (1.77828)(1.33352)(1.074607)(1.036633)(1.018152)(1.009035)(1.001643) \approx 2.7184,$$

与精确值 $$2.7183$$ 非常接近。

### 2.6 复数 (Complex Numbers) —— 22-5

即使推广到无理数，仍然有方程不可解：$$x^2 = -1$$。没有任何已发现的数——无论有理数还是无理数——的平方等于 $$-1$$。所以我们必须再次推广数系。

定义一个新数 $$i$$，使得 $$i^2 = -1$$。关于 $$i$$ 我们能说的基本就是这些。注意，$$x^2 = -1$$ 有两个根：$$i$$ 和 $$-i$$。任何包含 $$i$$ 的方程，将 $$i$$ 的符号处处取反后仍成立——这称为**取复共轭**。

按照所有规则将 $$i$$ 与实数组合后，发现所有数都可以写为形式:

$$p + iq,$$

其中 $$p$$ 和 $$q$$ 是我们此前定义的实数。$$i$$ 称为**虚数单位**，$$i$$ 的任意实数倍称为**纯虚数**。最一般的数 $$a = p + iq$$ 称为**复数**。

**复数乘法** (Eq. 22.4):

$$(r + is)(p + iq) = rp + r(iq) + (is)p + (is)(iq)$$
$$= rp + i(rq) + i(sp) + (ii)(sq)$$
$$= (rp - sq) + i(rq + sp).$$

因为 $$i^2 = -1$$。

**关键结论（代数基本定理的非正式表述）**: 仅凭引入 $$\sqrt{-1}$$ 这一项发明，**所有代数方程**都可以求解了！这是一个惊人的事实。费曼写道："最明显的猜测是我们将不得不一次又一次、一次又一次地发明新东西。但最大的奇迹是我们不必这样做。这是最后一项发明。"

**计算 $$10^{(r+is)}$$** (Eq. 22.5):

$$10^{(r+is)} = 10^r \cdot 10^{is}.$$

$$10^r$$ 我们已知如何计算，问题归结为计算 $$10^{is}$$。

令 $$10^{is} = x + iy$$。则其复共轭为 $$10^{-is} = x - iy$$。两者相乘:

$$10^{is} \cdot 10^{-is} = 10^0 = 1 = (x + iy)(x - iy) = x^2 + y^2. \tag{22.6}$$

所以 $$x^2 + y^2 = 1$$——找到了 $$y$$，就有了 $$x$$。

**如何计算 $$10^{is}$$**？关键假设：对于极小的 $$\epsilon$$，"定律" $$10^{\epsilon} = 1 + 2.3025\epsilon$$ 不仅对实数成立，对复数也成立。于是:

$$10^{is} \approx 1 + 2.3025 \cdot is, \quad s \to 0.$$

特别地，取 $$s = 1/1024$$:

$$10^{i/1024} = 1.00000 + 0.0022486i. \tag{22.7}$$

**表 22-3: $$10^{i/1024} = 1 + 0.0022486i$$ 的逐次平方**

|---|---|---|
| $$i/1024$$ | $$1$$ | $$1.00000 + 0.00225i$$ |
| $$i/512$$ | $$2$$ | $$1.00000 + 0.00450i$$ |
| $$i/256$$ | $$4$$ | $$0.99996 + 0.00900i$$ |

通过试错发现，将条目 "512" ($$i/2$$) 与 "128" ($$i/8$$) 相乘得到 $$0.13056 + 0.99159i$$，再乘 64、除以 4 和 2 等，最终得到纯粹虚数 $$i$$。总指数为:

$$i \cdot (512 + 128 + 64 - 4 - 2 + 0.20) / 1024 = i \cdot 698.20 / 1024.$$

因此:

$$\log_{10} i = 0.68184i.$$

我们看到了什么？$$x$$ 减小，穿过零，摆到接近 $$-1$$，又摆回来。$$y$$ 也在来回摆动。**这些数是周期性的**！$$10^{is}$$ 不断重复自身。

从 $$10^{0.68i} = i$$ 可知，其四次方 $$10^{2.72i} = i^4 = 1$$。因此 $$10^{is}$$ 具有周期性——当然，曲线看起来就像正弦和余弦，我们暂时称之为"代数正弦"和"代数余弦"。

**换底至 $$e$$**: 令 $$t = 2.3025s$$，则 $$10^{is} = e^{it}$$。于是:

$$e^{it} = \cos\_ t + i \sin\_ t. \tag{22.8}$$

这些函数具有什么性质？$$x^2 + y^2 = 1$$ 在底 $$e$$ 下同样成立，所以:

$$\cos\_^2 t + \sin\_^2 t = 1.$$

对于小 $$t$$，$$e^{it} = 1 + it$$，所以 $$\cos\_ t \approx 1$$，$$\sin\_ t \approx t$$——所有性质都与三角学中的正弦和余弦相同。

**周期**: $$i$$ 对底 $$e$$ 的对数是多少？对底 $$10$$ 是 $$0.68184i$$，乘以 $$2.3025$$ 得 $$1.570$$。所以这是"代数 $$\pi/2$$"——与通常的 $$\pi/2$$ 只差末位一位数字，这是计算误差造成的！

我们以纯代数的方式创造了两个新函数——余弦和正弦，它们属于代数，也**只**属于代数。然而最终醒来，却发现这些函数正是几何学中自然出现的函数。**所以归根到底，代数与几何之间存在联系。**

**欧拉公式——全章的高潮**:

$$e^{i\theta} = \cos\theta + i\sin\theta. \tag{22.9}$$

这就是那颗瑰宝。

**复数平面 (图 22-2)**: 将每个复数 $$x + iy$$ 表示为平面上的点——水平位置为 $$x$$，竖直位置为 $$y$$。如果该点到原点的径向距离为 $$r$$，角度为 $$\theta$$，则代数定律告诉我们:

$$x + iy = r e^{i\theta},$$

其中 $$x$$、$$y$$、$$r$$ 和 $$\theta$$ 之间的几何关系如图所示。这，就是**代数与几何的统一**。

## 三、关键示例与应用 (Key Examples)

### 3.1 用平方根表计算对数

这是"对数表的真正来源"。

**问题**: 求 $$\log_{10} 2$$。

**步骤**:
1. 注意到答案是介于 $$1/4$$ 和 $$1/2$$ 之间的某个值（因为 $$10^{0.25} \approx 1.778 \lt 2$$，而 $$10^{0.5} \approx 3.162 \gt 2$$）。
2. 从表中取出因子 $$10^{1/4} = 1.77828$$，从 $$2$$ 中除掉：$$2 / 1.77828 = 1.124682$$。这意味着我们从对数中移出了 $$0.250000$$。
3. 在表中找到略小于 $$1.124682$$ 的下一个数: $$1.074607$$（对应于 $$1/32$$ 即 $$32/1024$$）。除法给出 $$1.046598$$。
4. 继续用 $$1.036633$$（对应于 $$1/64$$ 即 $$16/1024$$）、$$1.0090350$$（对应于 $$1/256$$ 即 $$4/1024$$）依次除下去。
5. 最后剩下因子 $$1.000573$$，超出表的范围。利用近似公式：$$10^{\Delta/1024} \approx 1 + 2.3025\Delta/1024$$，得 $$\Delta = 0.254$$。
6. 总指数为: $$(256 + 32 + 16 + 4 + 0.254) / 1024 = 308.254 / 1024 = 0.30103$$。因此 $$\log_{10} 2 = 0.30103$$。

### 3.2 用平方表计算虚指数

**问题**: 求 $$\log_{10} i$$。

**步骤** (与计算实数对数完全平行，但处理复数):
1. 从 $$10^{i/1024} = 1.00000 + 0.0022486i$$ 出发（利用了 $$10^{i\epsilon} \approx 1 + 2.3025 i \epsilon$$ 的推广假设）。
2. 连续平方构造表 22-3。
3. 寻找表中数值的组合，使得乘积的实部为零而虚部为 $$1$$。
4. "512" ($$i/2$$) 与 "128" ($$i/8$$) 相乘得 $$0.13056 + 0.99159i$$。
5. 再乘 "64" ($$i/16$$) 得 $$-0.01308 + 1.00008i$$（过头了，实部变负）。
6. 除 "256" ($$i/4$$，通过乘共轭实现）继续调整。
7. 最终总指数为 $$i \cdot 698.20 / 1024$$，即 $$\log_{10} i = 0.68184i$$。
8. 换算为自然底: 乘以 $$2.3025$$ 得 $$1.570i$$——这正是 $$\pi/2$$ 的近似值。

### 3.3 计算自然底 $$e$$

**步骤**:
1. 已知 $$\log_{10} e = 1 / 2.3025\ldots = 0.434310\ldots$$
2. 转化为 $$444.73/1024 = (256 + 128 + 32 + 16 + 8 + 4 + 0.73)/1024$$。
3. 从表 22-1 取相应因子: $$1.77828, 1.33352, 1.074607, 1.036633, 1.018152, 1.009035, 1.001643$$（最后一项用近似公式计算 $$0.73/1024$$）。
4. 全部相乘得 $$2.7184$$，与真值 $$2.7183$$ 非常接近。

## 四、费曼的核心洞察 (Feynman's Core Insights)

### 4.1 抽象与推广——数学的真正驱动力

费曼在本章中反复展示一个统一的模式，它不仅适用于数的推广，也适用于整个理论物理学的方法论:

**规则 > 定义**。当我们面对一个在原有框架中不可解的问题时（如 $$3 - 5$$），我们不是放弃这个问题，而是保留代数规则 (22.1) 和 (22.2)，**将规则本身作为新符号的定义**。新符号不再有直观的"计数"含义——"$$-2$$ 乘以 $$5$$ 不能说成加 $$5$$ 加 $$-2$$ 次"——但它们在规则下运作无碍。然后我们再用规则推出对新符号的可计算结论（如 $$(-2) \times (-3) = 6$$）。

这就是抽象的本质：**从具体操作中抽离，以关系网络定义对象**。

### 4.2 "看起来仅仅是定义的东西，实际上涉及物理定律"

费曼强调了纯数学之外的哲学要点：在数系推广的每一步中，我们似乎只是在重新"定义"。从"相等速度意味着相等质量"到"如果两个质量各等于第三个质量，则它们彼此相等"——后者**不是**逻辑必然，而是**需要实验验证的物理定律**。数学推广本身虽然是形式操作，但它嵌入在经验的基础之上。

### 4.3 数值之美的"瑰宝"

尽管费曼说从物理学家观点出发，欧拉公式可以在两分钟内搬出来，但他选择绕道整个初等代数的大厦，最终以 $$e^{i\theta} = \cos\theta + i\sin\theta$$ 收束。这是他所谓的"智识的享受"——不是为了实用性，而是为了在宏大图景中看清每一块砖的位置。

**最深刻的惊喜**: 从纯粹代数出发（整数的计数、规则、逐次平方根），制造出两个代数函数（代数正弦和代数余弦），最终醒来却发现——它们正是几何学中的正弦和余弦。**代数与几何不是两个分离的世界，它们凭借复数这一桥梁而统一在 $$r e^{i\theta}$$ 之中。**

### 4.4 最后一个发明

从正整数到负整数、到有理数、到无理数、再到复数——费曼称之为"一次又一次地发明"。每当我们以为已经走完时，就会出现新的不可解方程。但引入 $$i = \sqrt{-1}$$ 之后，**所有**代数方程都可以求解了。这是代数的最终封闭性——代数学中的"代数基本定理"。费曼称之为 "the greatest miracle of all"（所有奇迹中最大的一个）: "我们不必再发明新东西了。"
