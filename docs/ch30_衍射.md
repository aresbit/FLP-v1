## 一、本章概要 (Overview)

- **核心问题**: 当一束光（或任何波动）遇到障碍物或通过狭缝时，它并不严格沿直线传播——光会"弯折"到几何阴影区内。如何用定量的数学语言描述这种弯折（即衍射）？光栅为什么能将白光分解为光谱色？我们能分辨两个角距离多近的光源？
- **逻辑链条**: $$n$$ 个等幅振子的干涉公式 (30-1) → 将这个公式推广为衍射光栅的角分布 (30-2) → 从光栅公式推导分辨本领的瑞利判据 (30-3) → 将同样的思想应用于抛物面天线和射电望远镜 (30-4) → 以薄膜颜色和 X 射线晶体衍射为例展示干涉/衍射的统一性 (30-5) → 以不透明屏边缘衍射为例，引入科纽螺线处理连续分布源 (30-6) → 最终推广到连续振荡电荷平面，揭示反射和折射的微观机制 (30-7)。
- **与前后章的关系**: 第29章讨论了干涉的一般原理，本章将其推广到连续分布的源——这就是"衍射"。费曼指出，"干涉"和"衍射"之间没有严格的物理分界线，只是习惯上的称谓差别。本章末尾对振荡电荷平面的处理是连接卷 I 波动光学与卷 II 电磁学的关键桥梁——它从麦克斯韦方程出发，展示了反射和折射定律如何从电荷的受迫振荡中自然地产生。

### 2.2 n 个等幅振子的合振幅 —— 30-1

本章是紧接着干涉那一章的继续。上一章我们讨论了多源干涉，本章我们将处理**连续分布**的源——这就是通常所说的"衍射"。事实上，在"干涉"和"衍射"两个词之间并没有重要的物理区别。我们通常说，当只有几个源（例如两个）时，结果是干涉。当有大量源时，这个词就往往变成衍射。我们习惯于这样用词，只要我们自己不糊涂，就没关系。

这一章将讨论许多干涉现象的推论，它们涉及将非常大量的源合在一起。它与上一处理若干源的章节不同。本章首先提及，有一个我们已经推导过的特殊公式，在许多地方被频繁使用。如果你有若干源，它们等间距地排成一行，并且每个源的信号都以某种方式到达，相位依次相差相同的量 $$\phi$$，则一个大家都熟悉的公式告诉我们任何角度的合振幅。我们在此重新推导一下。

假设有 $$n$$ 个源，所有源具有相同的振幅 $$A$$，但每个源的相位依次相差 $$\phi$$。则合振幅由下式给出：

$$R = A[\cos \omega t + \cos(\omega t + \phi) + \cos(\omega t + 2\phi) + \cdots + \cos(\omega t + (n-1)\phi)]. \tag{30.1}$$

这 $$n$$ 项的和可以在几何上这样求得：取长度均为 $$A$$ 的 $$n$$ 个矢量，每个矢量依次转动角度 $$\phi$$（图 30-1）。这些矢量构成一个等角多边形的边，其顶点落在一个圆上。该圆的半径 $$r = A / (2\sin(\phi/2))$$，而合振幅矢量的长度 $$A_R$$ 是弦长，其对应的圆心角为 $$n\phi$$。因此：

$$A_R = A\frac{\sin n\phi/2}{\sin \phi/2}. \tag{30.2}$$

合强度为振幅的平方，因此

$$I = I_0\frac{\sin^2 n\phi/2}{\sin^2 \phi/2}, \tag{30.3}$$

其中 $$I_0$$ 是单个源产生的强度。

这是一个伟大的公式——在本章中我们不断要用到它。

我们可以检查一下这个公式。对于 $$n = 1$$，式 (30.3) 给出 $$I = I_0$$。对于 $$n = 2$$，利用三角恒等式 $$\sin\phi = 2\sin(\phi/2)\cos(\phi/2)$$，我们得到 $$I = 4I_0\cos^2(\phi/2)$$，这正是我们上一章得到的结果。

费曼接着分析了当 $$n$$ 很大且 $$\phi$$ 很小时的行为。在 $$\phi = 0$$ 处，利用 $$\sin x \approx x$$，我们得到 $$I = n^2I_0$$。第一个极小出现在分子为零而分母不为零处，即 $$n\phi/2 = \pi$$，或

$$\phi = 2\pi/n. \tag{30.4}$$

在第一个极小之后，分子振盈，给出次极大。在 $$n\phi/2 = 3\pi/2$$ 附近，$$\sin^2$$ 几乎是 1，所以第一个次极大的强度约为 $$I_0 / \sin^2(3\pi/2n)$$。对于大的 $$n$$，$$\sin(3\pi/2n) \approx 3\pi/2n$$，因此第一个次极大的强度约为 $$n^2I_0 \times 4/(9\pi^2) \approx 0.045n^2I_0$$，即只有主极大的 $$4/(9\pi^2)$$，约 4.5%。第一个次极大显著低于中心极大。

将这一公式应用于一列排成一条直线的振荡电荷，相邻间距为 $$d$$。远场的相位差 $$\phi = \alpha + 2\pi d\sin\theta / \lambda$$，其中 $$\alpha$$ 是相邻源之间的本征相位差。观测方向与阵列法线的夹角为 $$\theta$$。

当下式成立时，强度达到极大：

$$\phi = 2\pi m \quad (m = 0, 1, 2, \ldots)$$

由此得到极大条件为

$$d\sin\theta = m\lambda \quad (m = 0, 1, 2, \ldots), \tag{30.6}$$

其中 $$m$$ 称为衍射级次。当 $$m = 0$$ 时，所有源同相，$$\theta = 0$$，即光束直射的方向。

最低极小发生在 $$\phi = 2\pi / n$$，意味着**从阵列两端出发的两条极限路径之间的总光程差等于一个波长** $$\lambda$$：

$$nd\sin\theta = \lambda, \tag{30.5}$$

这是一个非常简洁而重要的结果：对于长度为 $$L$$ 的阵列，强度在 $$\theta = \lambda / L$$ 处出现第一个零点。角宽度越窄，阵列越长。

### 2.3 衍射光栅 —— 30-2

在天线工作中，我们可以为所有的振荡器安排相同的相位。但在光学中，我们能否做到类似的事情？目前我们还不能实实在在地建造微型光频无线电台——带有无限细小导线的那种——但有一种非常容易的方法可以达到同样的效果。

假设我们有一套平行的细导线，间距为 $$d$$。远处一个射频源发出一个电场，电场在每个导线中驱动电子上下振荡。于是每个导线中的电子被相同的入射场以**相同的相位**驱动——每个导线随之产生一个新的辐射（这就是"散射"）。

在与阵列垂直的方向上，这些来自每个导线的散射波以相同的相位到达远处的接收器，强度达到极大——这就是透射光束。如果导线间距大于波长，还会在其他方向出现强散射，这些方向由

$$d\sin\theta = m\lambda$$

给出，即式 (30.6)。

同样的想法在光波情形中也起作用。在光波情形中，我们并不使用导线——导线必须极其微小以至于不可能制造——我们用的是划有**细小刻痕**的玻璃平板。把一块平整的玻璃板划上许多细小的平行凹痕，在凹痕处玻璃已被扰动，其散射方式不同于完整的玻璃，因此每条凹痕实际上当作一个独立的散射光源。如果把凹痕刻得非常精确地等间距排列——通常在每毫米内刻划几百条——我们就会得到一个顶顶漂亮的现象！光不仅直穿过去，在有限角度处也会出射一束强光，角度由 (30.6) 式决定，取决于刻痕间距和光的波长。这样的物体就叫做**衍射光栅**。

如果我们把用狭缝产生的一条很窄的光束照射光栅，除了中央的亮线外，在两侧还会各出现一个彩色光斑。这是因为 $$\theta$$ 依赖于波长 $$\lambda$$：对不同的颜色（即不同的波长），出射的角度不同！红光的波长在可见光中最长，因此出现在离中心最远的地方。此外，理论上我们还应能看到 $$m = 2$$ 的第二级光谱——尽管会非常微弱。

费曼指出，有趣的是，如果我们仔细制作光栅，刻痕无限细小且均匀分布，各级光谱的强度确实会是相等的。但现实中的光栅并非如此——连第一级左右两侧的强度都可能不相等。原因在于光栅被精心"设计"了：如果我们使用成对的散射体，每个散射体的强度不同且有相对相位，或者如果我们把刻痕刻成不对称的"锯齿"槽形，我们就可以把更多的光定向到某一特定的光谱级次中。"这看似复杂，实则是非常聪明的一种做法，因为这让光栅更加有用。"

接下来，费曼考虑了光以某一角度 $$\theta_{\text{in}}$$ 入射到光栅上的情形。相邻刻痕间入射波的相位差为 $$\alpha = -2\pi d \sin\theta_{\text{in}}/\lambda$$。于是相位差公式变为

$$\phi = \frac{2\pi d \sin\theta_{\text{out}}}{\lambda} - \frac{2\pi d \sin\theta_{\text{in}}}{\lambda}. \tag{30.7}$$

这里有一些有意思的事。对 $$m = 0$$，如果 $$d < \lambda/2$$，则 $$\vert\sin\theta_{\text{out}}\vert < 1$$ 的唯一解是 $$\sin\theta_{\text{out}} = \sin\theta_{\text{in}}$$。这意味着 $$\theta_{\text{out}}$$ 可以是 $$\theta_{\text{in}}$$ 的补角——也就是说，散射光沿入射光的方向继续前进，这就是**透射光束**。另一个解是 $$\theta_{\text{out}} = \theta_{\text{in}}$$——这就是我们所称的**反射光束**。

费曼对这一点作了极为深刻的物理评论："所以你会明白，光照射到一块反射材料上时，反射并不是简单的'光击中原子然后原子再发射一个新光波'——事情的要旨在于，当光照射到反射材料的原子上时，原子被驱动，原子再生出新的波，而当原子非常密集（间距远小于波长）时，散射光出射的方向恰好等于入射光的方向。"这就是说，反射定律——$$\theta_{\text{out}} = \theta_{\text{in}}$$——不是几何光学的"基本定律"，而是**从散射体的干涉中推导出来的**。这是本章的核心物理洞察之一。

最后，费曼分析了 $$d \to 0$$ 的连续极限：一段有限长度的固态材料中散射体无限密集。此时，相邻散射体之间的相位差趋近于零，但散射体的数目趋近于无穷大，总相位差 $$\Phi$$（从阵列的一端到另一端）保持为常数。令 $$\phi \to 0$$ 使得 $$\sin(\phi/2) \approx \phi/2$$，并将 $$n^2 I_0$$ 记作 $$I_m$$，式 (30.3) 变为

$$I = 4I_m \frac{\sin^2(\frac{1}{2}\Phi)}{\Phi^2}. \tag{30.8}$$

这个极限情形下，各级高级的光谱极大消失了——只剩下一个主极大，以及一些逐渐衰减的旁瓣。

### 2.4 光栅的分辨本领 —— 30-3

光栅的实用价值在于它能将不同波长的光分离到不同的角度。但是，如果两个波长非常接近，它们产生的衍射峰彼此重叠，光栅还能区分它们吗？——这就是**分辨本领**的问题。

费曼介绍了**瑞利判据**：当第一个波长的衍射图样的第一极小恰好落在第二个波长的衍射图样的极大上时，我们称这两个波长"刚好能被分辨"（图 30-6）。

对于 $$m$$ 级光谱，波长 $$\lambda' = \lambda + \Delta\lambda$$ 的 $$m$$ 级极大出现在

$$\Delta = mn\lambda'$$

处（$$\Delta$$ 是从光栅两端到观测点的总光程差）。波长 $$\lambda$$ 的 $$m$$ 级的第一极小出现在

$$\Delta = mn\lambda + \lambda$$

处。令这两个总光程差相等——这就是瑞利判据——我们得到

$$mn (\lambda + \Delta\lambda) = mn\lambda + \lambda.$$

化简后立即得到

$$\frac{\Delta\lambda}{\lambda} = \frac{1}{mn}, \tag{30.9}$$

或等价地

$$\frac{\lambda}{\Delta\lambda} = mn.$$

**分辨本领等于光栅总线数乘以所使用的光谱级次。** 这是一个极其简洁而优美的结果。

费曼还给出了一个更一般的表述。极限路径之间的**时间差**为

$$T = \frac{\Delta}{c} = \frac{mn\lambda}{c}.$$

因为频率 $$\nu = c/\lambda$$，我们有

$$\Delta\nu = \frac{c}{\lambda^2}\Delta\lambda.$$

结合式 (30.9)，我们得到

$$\Delta\nu = \frac{1}{T}. \tag{30.10}$$

这个公式不仅适用于光栅，而且适用于**任何仪器**——它是波动分析中的一个普适关系，等价于我们在瞬态那一章中得到的傅里叶带宽定理。

### 2.5 抛物面天线 —— 30-4

同样的分辨本领问题也出现在射电天文学中。射电望远镜通常用一个**抛物面反射器**来接收来自遥远射电源的信号。它的工作原理与光栅完全相同：电磁波到达抛物面的不同部分时走过的路径不同，当我们将来自整个抛物面的贡献叠加起来时，来自偏离中心方向的信号会相互抵消。

对于一个长度为 $$L$$ 的线阵天线，强度降到零的第一个角度由 $$L\sin\theta = \lambda$$ 给出，因此

$$\theta \approx \frac{\lambda}{L}$$

是能分辨的最小角度。费曼重点强调了**互易原理**：天线作为接收器时的方向图与其作为发射器时的方向图完全相同——物理定律在时间反演下是对称的。

对于**圆形**望远镜（孔径直径为 $$D$$），由于边缘处贡献的有效性略低，上式需要乘以一个 1.22 的因子：

$$\theta = 1.22\frac{\lambda}{D}.$$

费曼在脚注中指出，瑞利判据本身只是一个粗略的判据——如果你仔细测量强度分布，你实际上可以分辨比这个极限还要近得多的两个源。

### 2.6 彩色薄膜；晶体 —— 30-5

本节将干涉/衍射的概念应用于两个日常生活中的例子。

**薄膜颜色**：油膜、肥皂泡或者湿路面上的汽油呈现绚丽的彩色。这是因为光在薄膜的上表面和下表面发生反射，两个反射波之间的相位差依赖于波长、薄膜的厚度以及观察的角度。当膜厚或角度合适时，某种波长的光被加强，另一种被削弱——于是我们看到了颜色随着角度变化。

**晶体中的 X 射线衍射**：这与光栅的原理完全相同，只是用晶体的原子层代替了刻痕。当X射线照射到晶体上时，满足布拉格条件 $$2d\sin\theta = m\lambda$$ 的方向会出现相干加强（即"反射"）。费曼指出，通过测量不同级次的反射强度，我们可以反推出晶体内部原子的具体排列方式——这是一场漂亮的"侦探工作"，它催生了整个 X 射线晶体学领域。

### 2.7 不透明屏的衍射 —— 30-6

现在我们来处理一个看起来更复杂的问题：光经过不透明屏的直边时会发生什么？——在几何阴影的边界附近会出现明暗交替的条纹（图 30-7）。

与之前处理的线性相位差（$$\phi \propto \theta$$）不同，现在的问题在于，从屏边上方高度 $$h$$ 处的点到达观测点 $$P$$ 的相位差是一个二次函数：

$$\Delta\phi \propto \frac{h^2}{2s},$$

其中 $$s$$ 是观测屏到障碍物的距离。这意味着相位差不是随高度线性增加的，而是**抛物线式**增加的。

为了将这些贡献加起来，费曼引入了**科纽螺线**（图 30-8）——这是复平面中的一条曲线，沿曲线的弧长 $$v$$ 正比于高度 $$h$$，而每一点的切线角正比于 $$v^2$$。从曲线的一端到另一端画出矢量就给出了合振幅。

利用科纽螺线，我们可以读出不同位置处的光强：

- **远远进入亮区**：矢量从螺线的一端延伸到另一端，振幅最大——归一化为 1。
- **恰好位于几何阴影边界处**：矢量从螺线的拐点（中点）延伸到一端——振幅只有全振幅的一半，因此**强度只有自由场的 1/4**。这是一个精确的结果：**$$I_{\text{边}} = \frac{1}{4} I_0$$**。
- **进入亮区时**：矢量的一端沿螺线的一个分支滑动，产生振幅的振荡——这就是衍射条纹的成因。
- **进入暗区时**：矢量缩短，强度单调下降为零。

图 30-9 展示了强度作为位置函数的曲线，在亮区一侧振荡，在暗区单调衰减。

### 2.8 振荡电荷平面的场 —— 30-7

本节是本章的理论巅峰。费曼从一个最根本的视角来处理问题：考虑在 $$xy$$ 平面内均匀分布的面密度为 $$\eta$$（单位面积上的电荷数）的振荡电荷，每个电荷的电荷量为 $$q$$，所有电荷以相同的相位振动，位移为 $$x_0 e^{i\omega t}$$。观察点 $$P$$ 位于 $$z$$ 轴上的距离 $$z$$ 处（图 30-10）。

从单个电荷在 $$Q$$ 处产生的辐射场出发，其加速度为 $$a = -\omega^2 x_0 e^{i\omega t}$$，经过投影（在远场近似下，投影因子近似为 1），在 $$P$$ 点的电场为：

$$E_{\text{来自 }Q} \approx \frac{q}{4\pi\epsilon_0 c^2} \cdot \frac{\omega^2 x_0 e^{i\omega(t - r/c)}}{r}. \tag{30.11}$$

将整个平面上所有电荷的贡献加起来，我们需要计算

$$E_{\text{总}} = \frac{q}{4\pi\epsilon_0 c^2}\, \omega^2 x_0 e^{i\omega t}\, \eta \int_0^\infty \frac{e^{-i\omega r/c}}{r}\, 2\pi\rho\, d\rho. \tag{30.12}$$

核心积分是

$$\int_{\rho=0}^{\rho=\infty} \frac{e^{-i\omega r/c}}{r}\,\rho\, d\rho. \tag{30.13}$$

利用几何关系 $$r^2 = \rho^2 + z^2$$，微分得 $$2r\,dr = 2\rho\,d\rho$$，也就是 $$\rho\,d\rho = r\,dr$$。这是一个关键的简化——代换后分母中的 $$r$$ 被消去了：

$$\int_{r=z}^{r=\infty} e^{-i\omega r/c}\, dr. \tag{30.15}$$

直接积分得到

$$-\frac{c}{i\omega} \left[e^{-i\infty} - e^{-(i\omega/c)z}\right]. \tag{30.16}$$

$$e^{-i\infty}$$ 这个项在数学上是未定义的——费曼以他标志性的方式给出了物理解释。他把积分看作在复平面中逐段相加许多矢量，每个矢量长 $$\Delta r$$，角度为 $$-\omega r/c$$。这些矢量排成一个**圆**（图 30-11）。一直积分到无穷，就是在圆上无限循环——这在数学上是未定的。

但是物理上，面电荷的分布不可能是真正无限的——它必须在某个很大的 $$\rho$$ 处逐渐衰减。每一圈新的矢量贡献都比上一圈稍短，因此矢量之和并不是沿圆周无限循环，而是**螺旋收敛到圆的中心**（图 30-12）。因此物理上正确的积分结果就是：从起点到圆心的矢量。数学上等价于令 $$e^{-i\infty} = 0$$，结果即为

$$\frac{c}{i\omega} e^{-i\omega z/c}. \tag{30.17}$$

将所有因子代回，我们得到在 $$P$$ 点的总电场：

$$E_{\text{总在 }P} = -\frac{\eta q}{2\epsilon_0 c}\, i\omega x_0\, e^{i\omega(t - z/c)}. \tag{30.18}$$

注意到 $$i\omega x_0 e^{i\omega t}$$ 正是电荷的**速度**，我们可以将上式重写为

$$E_{\text{总在 }P} = -\frac{\eta q}{2\epsilon_0 c}\,[\text{电荷的速度}]_{\text{在 } t - z/c}. \tag{30.19}$$

这个结果的深刻之处在于：

1. **推迟时间仅依赖于垂直距离** $$z$$——这是到平面的最短距离，而非到每个电荷的实际路径长度。那些"迂回"路径的贡献在无穷积分中恰好抵消掉了。
2. 这个公式**对任意 $$z$$ 都成立**，甚至当 $$z < \lambda$$ 时也是如此——它是一个精确结果。
3. 这个结果是从一个"微观"模型（每个电荷独立地辐射）推导出来的，但它最终给出了一个"宏观"的波动行为——平面波在空间中传播，其场只依赖于电荷在推迟时刻的速度。

在卷 II 中，费曼将用这个结果作为出发点，来**推导反射定律和折射定律**——展示它们不过是电荷在入射场驱动下所产生的相干辐射的必然结果。本节堪称从波动光学过渡到电磁理论的点睛之笔。

## 三、延伸思考与关键启示 (Further Reflections)

### 3.1 干涉与衍射的统一

费曼在本章开头就指出"干涉"和"衍射"之间没有本质的物理区别。回顾全章，我们一直在使用**完全相同的数学公式**——式 (30.3) 中的 $$\sin^2(n\phi/2)/\sin^2(\phi/2)$$——来处理两个源（干涉）和无穷多个源（衍射）的情形。变化的只是 $$n$$ 的大小和 $$\phi$$ 的含义。这个数学上的连续性反映了物理上的统一性：无论源的数量是多少，基本原理始终是把**复振幅在复平面中加起来**。

### 3.2 从离散到连续的桥梁——科纽螺线

科纽螺线（30-6 节）是物理学中"从离散到连续过渡"这一思想的优雅体现。当源离散分布时，我们把等长的矢量依次相加；当源连续分布时，矢量的长度变为无穷小，其方向连续旋转——这就产生了在复平面中的一条光滑曲线。科纽螺线的参数方程——曲线弧长 $$v$$ 和切线角 $$\pi v^2/2$$ 之间的二次关系——精确地编码了本节中的物理：相位差与距离的平方成正比。

### 3.3 微观→宏观——从散射电荷到反射定律

30-7 节是费曼物理观的集中体现。传统的几何光学将反射定律（入射角等于反射角）当作一个基本经验定律。费曼告诉我们：不，反射定律是**推导出来的**。它来自于一个更基本的图景——每个原子在入射光的驱动下振荡，产生自己的辐射场；当无穷多个这样的原子密集地排列在一个平面上时，它们的辐射场在除了反射角以外的所有方向上通过干涉相消，**只剩下反射角的方向上相干加强**。

这个论证的威力在于：当你理解了这个机制，你就会明白为什么反射镜不需要是完美的导体——电阻损耗只会使反射不够完美，但不会改变反射角。你也明白了为什么即使是非金属表面（如水面、玻璃面）也会反射光——因为原子中的束缚电子也会在电场中被驱动振荡。

### 3.4 分辨本领与不确定性原理

式 (30.10) —— $$\Delta\nu = 1/T$$ —— 是信息论中时间-频率不确定性关系在波动光学中的体现。光栅的物理长度决定了波列到达光栅不同部分的时间色散 $$T$$；正是这个时间色散，使得光栅像一个傅里叶分析仪：时间色散越大，频率分辨得越精细。这一思想在量子力学中以 $$\Delta E \cdot \Delta t \sim \hbar$$ 的形式再次出现，展示了经典波动光学与量子力学之间深刻的类比关系。

## 四、复习与讨论题 (Review and Discussion)

1. 费曼在本章开头指出干涉和衍射之间没有本质的物理区别。回顾第29章（双缝干涉）和本章（光栅衍射），这两种现象使用了完全相同的数学公式 $$\sin^2(n\phi/2)/\sin^2(\phi/2)$$。当 $$n=2$$ 时我们称为干涉，当 $$n$$ 很大时我们称为衍射。那么在 $$n=3, 4, 5, \ldots$$ 的中间区域，我们应该使用哪个词？这种命名惯例是否反映了某种更深层的物理转变，还是纯粹的历史偶然？

2. 费曼在 30-2 节中论证了反射定律可以从散射体的干涉中推导出来——反射不是一个"基本定律"，而是"推导出来的"。这与我们在基础物理课程中学到的"反射定律是一个基本经验定律"形成鲜明对比。你如何看待这种差异？"基本"一词在不同的物理层次（几何光学→波动光学→电磁理论→量子电动力学）中是否有不同的含义？

3. 在 30-6 节中，科纽螺线被用来处理直边衍射。如果不使用科纽螺线，只用矢量多边形的方法——即把直边上方离散地分为许多间距为 $$\Delta h$$ 的小段——会面临什么问题？科纽螺线作为连续极限，相比于离散求和的优势在哪里？

4. 在 30-7 节中，费曼用了这样一个论证：对于平面上的无限电荷阵列，对 $$e^{-i\omega r/c}$$ 从 $$r=z$$ 积分到 $$r=\infty$$ 时，他令上界处的 $$e^{-i\infty} = 0$$。他给出的物理论证是：实际上不存在真正的无限平面——远处的电荷分布的逐渐衰减导致向量的螺旋收敛。这个论证类似于处理发散积分时常用的"物理截断"。你能想到物理学中另一个使用类似论证（将数学上未定义的项设为零，因为物理上系统总有一定程度的不完美）的例子吗？

5. 式 (30.10) 给出的 $$\Delta\nu = 1/T$$ 是一个普适关系。对于一个长度为 $$L$$ 的光栅来说，$$T = (L\sin\theta)/c$$。假设你想让一个光栅能分辨钠双线（589.0 nm 和 589.6 nm），在最低的 $$m=1$$ 级光谱上至少需要多少条刻线？如果每毫米刻 600 条线，光栅的总宽度至少应该是多少？
