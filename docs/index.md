## 关于本书

《费曼物理学讲义》是 20 世纪最具影响力的物理学教科书。1961-1963 年，理查德·费曼在加州理工学院为本科生讲授了两年的物理学导论课程，这些讲义由 Robert B. Leighton 和 Matthew Sands 整理出版。

**卷1** 涵盖：力学、相对论、旋转、振动与波、光学、电磁辐射、量子行为、统计力学、热力学、声学，以及作为终章的物理定律中的对称性。

全书以费曼标志性的口语化风格写成——清晰、机智、充满物理直觉。费曼从不满足于"正确"的解释，他追求的是那种让你忍不住说"原来如此！"的洞察。

> "我们所需的是想象力，但必须是套在可怕紧身衣里的想象力。"
> — Richard Feynman

## 翻译说明

- **风格**: 忠实保留费曼的口语化、诙谐的演讲风格，同时确保专业术语的准确性
- **术语**: 首次出现标注 `中文 (English)`，全书术语统一
- **数学**: 全部使用 `$$...$$` 格式，兼容 GitHub Pages (kramdown + MathJax 3)
- **公式**: 经 kramdown 验证，零个公式泄露、零个虚假表格、零个强调损坏
